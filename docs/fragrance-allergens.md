# Canadian Fragrance Allergen Reference (24 + 81)

Reference data for the Canadian cosmetic fragrance allergen disclosure
requirement. Use to audit and expand the `ingredients` table so the
`is_fragrance_allergen` flag is accurate for each phase.

## Regulatory source

[Health Canada — Cosmetic advertising, labelling and ingredients](https://www.canada.ca/en/health-canada/services/cosmetics/cosmetic-advertising-labelling-ingredients.html#s4)
implements EU Regulation 2023/1545 (Annex III amendment).

| Phase | When | Coverage |
|---|---|---|
| Phase 1 | **April 12, 2026** (in effect) | 24 allergens, all cosmetics |
| Phase 2 | **August 1, 2026** | 81 allergens, newly-introduced products |
| Phase 3 | **August 1, 2028** | 81 allergens, all products on the market |

## Disclosure thresholds

- **Leave-on products** (lotion, cream, balm, perfume, lip products): >0.001% (10 ppm)
- **Rinse-off products** (shampoo, body wash, soap, scrub): >0.01% (100 ppm)

## Phase 2 — the 57 added allergens (August 1, 2026)

These bring the total to 81. Common ones grouped by source family:

### Pinene / terpene family
- alpha-Pinene (CAS 80-56-8) — pine, fir, frankincense
- beta-Pinene (CAS 127-91-3) — pine, rosemary
- Camphor (CAS 76-22-2) — rosemary, white camphor, lavender (trace)
- Carvone (CAS 99-49-0) — caraway, dill, spearmint
- 3-Carene (CAS 13466-78-9) — pine
- Terpinolene (CAS 586-62-9) — pine, citrus
- alpha-Terpinene (CAS 99-86-5) — eucalyptus, tea tree
- gamma-Terpinene (CAS 99-85-4) — citrus, tea tree

### Citrus / lemon family
- Citronellal (CAS 106-23-0) — lemongrass, citronella

- Linalyl acetate (CAS 115-95-7) — lavender, bergamot
- Hexyl salicylate (CAS 6259-76-3) — synthetic
- Trimethylbenzenepropanol (CAS 103694-68-4) — synthetic floral

### Mint / camphoraceous
- Menthol (CAS 89-78-1, 1490-04-6, 2216-51-5) — peppermint, spearmint
- 1,8-Cineole / Eucalyptol (CAS 470-82-6) — eucalyptus, rosemary

### Florals (not in the original 24)

## Audit your current `ingredients` table

Run these in Supabase SQL Editor to see where you stand.

### How many ingredients are currently flagged?

```sql
SELECT
  COUNT(*) FILTER (WHERE is_fragrance_allergen) AS flagged_allergens,
  COUNT(*) AS total_ingredients
FROM public.ingredients;
```

### Which Phase 1 (24) allergens are present?

```sql
SELECT inci_name, common_name, is_fragrance_allergen
FROM public.ingredients
WHERE LOWER(inci_name) IN (
  'amyl cinnamal',
  'amylcinnamyl alcohol',
  'anise alcohol',
  'anisyl alcohol',
  'benzyl alcohol',
  'benzyl benzoate',
  'benzyl cinnamate',
  'benzyl salicylate',
  'cinnamal',
  'cinnamaldehyde',
  'cinnamyl alcohol',
  'citral',
  'citronellol',
  'coumarin',
  'eugenol',
  'farnesol',
  'geraniol',
  'hexyl cinnamal',
  'hydroxycitronellal',
  'isoeugenol',
  'limonene',
  'linalool',
  'methyl 2-octynoate',
  'methyl heptine carbonate',
  'alpha-isomethyl ionone',
  'evernia furfuracea (treemoss) extract',
  'evernia prunastri (oakmoss) extract'
)
ORDER BY inci_name;
```

### Which Phase 1 allergens are MISSING from the DB?

Run the query above. Compare the returned list against the 24 in this
doc. Any name in the doc but not in the result needs to be inserted.

### Which present rows are NOT yet flagged?

```sql
UPDATE public.ingredients
SET is_fragrance_allergen = TRUE,
    updated_at = NOW()
WHERE LOWER(inci_name) IN (
  -- paste the 24 names from above
  'amyl cinnamal', 'benzyl alcohol', 'benzyl benzoate',
  'benzyl salicylate', 'cinnamal', 'cinnamyl alcohol',
  'citral', 'citronellol', 'coumarin', 'eugenol',
  'farnesol', 'geraniol', 'hexyl cinnamal',
  'hydroxycitronellal', 'isoeugenol', 'limonene',
  'linalool', 'alpha-isomethyl ionone',
  'evernia furfuracea (treemoss) extract',
  'evernia prunastri (oakmoss) extract'
)
AND is_fragrance_allergen = FALSE;
```

### Add missing allergens

For each one in the doc that's not in the DB, run:

```sql
INSERT INTO public.ingredients (
  inci_name, common_name, cas_number,
  hotlist_status, is_fragrance_allergen,
  description, slug
)
VALUES (
  'Linalool', 'Linalool',
  '78-70-6',
  'not_listed', TRUE,
  'A naturally-occurring fragrance allergen found in lavender, coriander, and rosewood essential oils. Must be individually disclosed on Canadian cosmetic labels above 0.001% (leave-on) or 0.01% (rinse-off) per Health Canada''s April 12, 2026 fragrance allergen rule.',
  'linalool'
)
ON CONFLICT (inci_name) DO UPDATE
SET is_fragrance_allergen = TRUE,
    updated_at = NOW();
```

Adjust per-ingredient. Use lowercase-hyphenated for `slug`.

## Phase 2 expansion (August 1, 2026) — DB to-do

When you're ready to flag the full 81-allergen list:

1. Source the complete list from [EU Regulation 2023/1545](https://eur-lex.europa.eu/eli/reg/2023/1545/oj) Annex III.
2. Cross-reference against the 24 already in this doc.
3. Insert/update each new allergen in `ingredients` following the Phase 1 pattern above.
4. The CNF Readiness Checker and label-tab fragrance allergen disclosure logic auto-detect via `is_fragrance_allergen = TRUE`, so no app code changes are required.

The 57 additions span pinenes, cedrols, ionones, damascones, and several
synthetic musks — many are common in natural EOs and synthetic FOs that
indie makers use today, so this expansion has real impact.
